# Mail Sphere Project

Separate frontend (Vue 3 + Vite) and backend (Node + Express) with a basic login page.

## Requirements
- Node.js 18+

## Quick start

### Backend
```bash
cd backend
npm install
npm run dev
```
Backend runs on http://localhost:3001

### Frontend
```bash
cd frontend
npm install
npm run dev
```
Frontend runs on http://localhost:5173
